This folder contains all codes for the final analysis.

To prepare data for the interim analysis:
1) Run interim_01_import.R
2) Run interim_02_clean.R
3) Run interim_03_createVAD.R
(.Rdata files should be saved in Data and InterimAnalysis folders at the top level)

To prepare data for the final analysis:
1) Run 01_import.R
2) Run 02_clean.R
3) Run 03_createVAD.R
(.Rdata files should be saved in Data at the top level)

To reproduce the final analysis:
1) Run 04_analysis.R
2) Run 05_report.Rmd
(figures should be saved in subfolder figures)

Additional functions/libraries should be saved in subfolder functions
